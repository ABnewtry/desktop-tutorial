# 🎬 دستور التصميم السينمائي التفاعلي — AB Cinematic Playbook
> المرجع الكامل لنوعية **Scroll-Driven Cinematic WebGL** — كل أداة يمكن تطبيقها، متى تستخدمها، وكيف تطلبها.
> النسخة 1.0 — إعداد: استوديو AB

---

## 1. الكاميرا (Camera) — عين المخرج

| الأداة | الوصف | متى تستخدمها |
|---|---|---|
| **Camera Path (CatmullRom)** | منحنى ثلاثي الأبعاد تمشي عليه الكاميرا مع التمرير | العمود الفقري لأي فيلم تمرير |
| **Wide Shot** | لقطة واسعة z بعيد — الكائن صغير في فضاء | الافتتاح: الرهبة |
| **Macro Shot** | قريب جداً على تفصيلة | إبراز الحِرفة والخامة |
| **Orbit** | دوران حول الكائن | استعراض القطعة من كل زاوية |
| **Dolly Push-in** | تقدّم مستقيم نحو الكائن | بناء توتر تدريجي |
| **Dolly Zoom (Vertigo)** | الكاميرا ثابتة تقريباً والعدسة FOV تتغير | إحساس دوار/انبهار — لقطة هيتشكوك |
| **Crane / Rise** | صعود أو هبوط رأسي | البداية من الأرض للبطل، أو الغطس |
| **Dutch Angle (Roll)** | إمالة الكاميرا على محورها Z | توتر، حداثة، كسر الرتابة |
| **Look-at Offset** | نقطة نظر الكاميرا تنزاح عن المركز | ترك مساحة للنص (rule of thirds) |
| **Mouse Parallax** | الكاميرا تميل قليلاً مع حركة الماوس | عمق حي حتى بدون تمرير |
| **FOV** | 28–35 تليفوتو سينمائي · 50+ عدسة واسعة مشوّهة | 35 هو الذهبي؛ تجاوزه بقصد فقط |
| **Camera Shake** | اهتزاز ضئيل جداً (نويز) | لحظة اصطدام/قوة — بحذر شديد |

## 2. الإضاءة (Lighting) — 90% من الفخامة

| الأداة | الوصف | القيمة الذهبية |
|---|---|---|
| **Key Light** (Directional) | الضوء الرئيسي المشكّل للحجم | intensity 1.5–2.2 من زاوية 45° |
| **Rim / Kicker** (Spot خلفي) | حافة مضيئة تفصل الكائن عن الخلفية | بلون التمييز — أهم ضوء |
| **Rect Area Light** | شريط انعكاس طويل ينزلق على الأسطح | 8×0.6 — سر إعلانات الساعات |
| **Ambient** | ضوء عام خافت يمنع السواد الميت | 0.10–0.20 لا أكثر |
| **HDRI / Environment** | بيئة انعكاسات — بدونها المعدن بلاستيك | إلزامي لأي خامة عاكسة |
| **Practical Glow (Emissive)** | جزء من الكائن يضيء ذاتياً | الحفر، الشعارات، الخطوط |
| **Light Choreography** | شدة/موضع الأضواء تتغير مع التمرير | الإضاءة تحكي فصول القصة |
| **God Rays** | أعمدة ضوء حجمية | مشاهد الغبار والدخان |
| **Caustics (محاكاة)** | تموّجات ضوء الماء | ثيمات الأعماق والزجاج |

## 3. الخامات (Materials) — جلد الكائن

| الخامة | المفاتيح | الشخصية |
|---|---|---|
| **Polished Metal** | metalness 1 / roughness .15–.25 | كروم، ذهب، فضة — فخامة كلاسيكية |
| **Brushed Metal** | metalness 1 / roughness .35–.5 | صناعي راقٍ، هادئ |
| **Glass / Transmission** | transmission 1 / ior 1.5 / thickness | نقاء، عطور، كريستال |
| **Iridescence** | iridescence 1 + IOR | تقزّح لؤلؤي — ألوان تتحرك مع الزاوية |
| **Fabric / Matte** | metalness 0 / roughness .85+ | قماش، دفء، إنسانية |
| **Obsidian / Gloss Black** | أسود لامع roughness .15 | غموض، مونوليث، حداثة صارمة |
| **Emissive Core** | emissive بلون التمييز + Bloom | نيون، طاقة، حياة داخلية |
| **Liquid Chrome** | معدن + تشوه سطح متحرك | مستقبلي، عضوي، Y2K |
| **Clearcoat** | طبقة ورنيش فوق الخامة | سيارات، بيانو، فخامة مضاعفة |
| **Wireframe** | الهيكل السلكي فقط | تقني، blueprint، ما قبل الولادة |

## 4. طبقة السينما (Post-Processing) — ما بعد التصوير

| الأداة | الوصف | الجرعة |
|---|---|---|
| **Bloom** | هالة حول المضيء | 0.3–0.6 · فوقها = حريق |
| **Film Grain / Noise** | حبيبات الفيلم | 0.02–0.04 — الفرق بين رندر ولقطة |
| **Vignette** | تعتيم الأطراف | 0.4–0.6 يركّز العين للمركز |
| **Depth of Field (DOF)** | عمق ميدان — خلفية ضبابية bokeh | لقطات الماكرو فقط |
| **Chromatic Aberration / RGB Shift** | انزياح ألوان طفيف عند الحواف | 0.0004 عدسة حقيقية · أعلى = غليتش |
| **Afterimage / Motion Trails** | أشباح الحركة تذوب ببطء | سرعة، سيولة، حلم |
| **Fog** | ضباب يبلع البعيد | عمق لا نهائي بثمن رخيص |
| **Tone Mapping (ACES)** | معايرة الألوان السينمائية | ACESFilmic دائماً + exposure 0.85–1.0 |
| **Scanlines / CRT** | خطوط شاشة قديمة | ثيمات retro-tech فقط |
| **Pixelation / Dither** | تكسير البكسل المتعمد | لحظات انتقال أو هوية 8-bit |

## 5. الحركة (Motion) — قواعد الزمن

| الأداة | الوصف | القيمة الذهبية |
|---|---|---|
| **Scrub** | تأخر الفيلم عن الإصبع | 1–1.5 · أبداً true |
| **Lenis Smooth Scroll** | نعومة التمرير نفسها | lerp 0.08 — إلزامي |
| **Easing** | منحنى كل حركة | power3.out للدخول · لا linear أبداً |
| **Text Reveal (clip)** | السطر يصعد من قناع | duration 1.1 · stagger 0.08 |
| **Scramble / Decode** | الحروف تتشفّر ثم تتضح | عناوين تقنية، أرقام |
| **Counter / Odometer** | الأرقام تعدّ لقيمتها | إحصاءات، preloader |
| **Idle Float** | تنفس الكائن وهو ساكن | sin بطيء ±0.05 |
| **Scroll Velocity FX** | سرعة تمريرك تغذّي تأثيراً (ميل/تشوه) | إحساس فيزيائي حي |
| **Magnetic Elements** | الزر ينجذب للمؤشر | نصف قطر 80–120px |
| **Custom Cursor** | نقطة تتبع بـ lerp 0.15 وتتضخم فوق التفاعلي | توقيع النوعية |
| **Preloader Counter** | 00→100 ثم كشف clip-path | البوابة الإلزامية للفيلم |
| **Section Pinning** | قسم يثبت والمحتوى يتبدل داخله | مقارنات، تسلسلات |

## 7. الصوت البصري (Typography as FX)
- **Outline Text**: حروف مفرّغة stroke فقط — تحتها المشهد يبان.
- **Ghost Numerals**: أرقام عملاقة شفافة خلف المحتوى.
- **Kinetic Type**: كلمة تتحرك مع التمرير (تتمدد/تدور).
- **Split Hero**: سطر ممتلئ + سطر مفرّغ = إيقاع.
- **Label System**: 12px uppercase letter-spacing 0.18em — توقيع الفخامة.

## 8. قواعد الدستور الذهبية
1. كائن بطل **واحد** — أي شريك يقتل الهيبة.
2. لون تمييز **واحد** — الثاني يفقدك الفخامة فوراً.
3. لا linear easing — أبداً.
4. لا #000 صافي ولا #FFF صافي.
5. Grain دائماً — بدونه "رندر" مو "لقطة".
6. النص 3–7 كلمات لكل لقطة.
7. fov 35 إلا بقصد درامي.
8. البريلودر ليس زينة — هو ستارة المسرح.
