# بورتفوليو عبد الرحمن (AB) — ملف ذاكرة المشروع

> اقرأ هذا الملف كاملاً قبل أي تعديل. هو المرجع الرسمي للمشروع.

## نظرة عامة
- **المشروع:** بورتفوليو شخصي لـ"عبد الرحمن / Abdulrahman" — خبير بالذكاء الاصطناعي (تسويق، تصميم إعلانات وبوسترات، ستوري بورد، لاندنق بيجز، هويات بصرية، مونتاج Premiere Pro).
- **الموقع الحي:** https://abnewtry.github.io/desktop-tutorial/
- **الفرع الرسمي للنشر:** `claude/kind-cannon-mm71aj` — GitHub Pages ينشر من هذا الفرع (root). **كل تعديل يُدفع له مباشرة.**
- **البنية:** ملف واحد `index.html` (~4MB) يحوي كل شيء: CSS + JS + كل الصور مدمجة base64. لا build ولا اعتماديات.

## أسلوب العمل المتفق عليه مع عبد الرحمن
1. **خطوة بخطوة** — تعديل واحد، معاينة، موافقته، ثم التالي. لا تنفذ أشياء كثيرة دون سؤاله.
2. **رفع الصور:** يرفعها هو عبر رابط `https://github.com/ABnewtry/desktop-tutorial/upload/claude/kind-cannon-mm71aj/assets/uploads` — أحياناً تروح غلط على فرع `main`، فابحث عنها في الفرعين (`git ls-tree origin/main`).
3. **معالجة الصور:** Pillow (`pip install Pillow`) → تصغير + JPEG q80 → base64 → دمج في index.html → `git rm` للأصل الثقيل من الريبو.
4. **بعد كل تعديل:** تحقق JS (`node --check` على محتوى `<script>`)، commit + push، أرسل له معاينة index.html عبر SendUserFile، وذكّره أن الموقع يتحدث خلال 1-3 دقائق.
5. **لو قال "التعديل ما ظهر":** افحص GitHub Actions (`pages build and deployment`) — أحياناً يفشل البناء عشوائياً؛ الحل: commit فارغ وpush لإعادة الإطلاق. وأعطه الرابط مع `?v=رقم` لكسر الكاش.
6. **الكتابة:** عربي سعودي قصير + نبرة "بروفيسور علم نفس تسويقي" — جمل قصيرة، تفاصيل محسوسة، كلمات مفتاحية بتوهيج `<b class='hl'>`. كل نص له نسخة EN مطابقة.

## أفكار اتجاهات بديلة للفيلم السينمائي (محفوظة — قد نحتاجها، مرتبة بالأولوية)
> طلبها بتاريخ 2026-07-10: «4 أفكار مختلفة تماماً على نمط هذا التفكير، احفظها ورتّبها».
1. **«استوديو المخرج / THE STUDIO»** — بدل الفضاء: استوديو تصوير ثلاثي الأبعاد. الكائن البطل عدسة كاميرا سينمائية عملاقة (زجاج + انعكاسات)، حولها سوفت بوكس وأضواء عملية. كل فصل «سيت» تصوير: أعماله تظهر كشاشات مونيتور وبوسترات معلقة داخل الاستوديو، والكاميرا تمشي Dolly حقيقي بين السيتات. يبيع هوية «مصمم + مونتير بريمير» مباشرة.
2. **«الممر المعماري / THE GALLERY»** — متحف ليلي فخم: الكاميرا تتقدم للأمام دائماً عبر أقواس/بوابات، كل فصل قاعة بإضاءتها الخاصة، الأعمال براويز ضخمة مضاءة Spot على الجدران، أرضية رخام سوداء عاكسة (Reflector). أفخم إحساس ممكن — «معرض أعمال» بالمعنى الحرفي.
3. **«التحوّل / METAMORPHOSIS»** — كائن واحد يتحوّل (Morphing) مع كل فصل: يبدأ نقطة ضوء → يتجمع صورة → يتشكل بوستر → ينبسط صفحة لاندنق → ينطوي شعاراً → يستقر نواة ذكاء. الكائن نفسه يحكي الخدمات بجسده. الأصعب تقنياً والأكثر إبهاراً — يحتاج جلسة كاملة.
4. **«شريط الفيلم / FILMSTRIP 35mm»** — التمرير يجرّ شريط فيلم 35mm عملاقاً فيه كوادر أعماله، الكاميرا Macro على الكوادر، ثقوب الشريط وعدّاد Timecode يجريان، ضوء بروجيكتور خلفي وغبار عالق في شعاعه. يبيع هوية المونتاج، ونوستالجيا سينمائية صرفة.

## ab-film.html — البورتفوليو السينمائي v3 (فيلم تمرير كامل)
- صفحة مستقلة: https://abnewtry.github.io/desktop-tutorial/ab-film.html — **لا تعدّل index.html من أجلها**، نسخة سينمائية موازية بكامل معلومات البورتفوليو.
- **v2 بناءً على ملاحظاته:** (1) عربي فصيح مدقق بلا أخطاء أحرف، (2) ألوان البورتفوليو نفسها (بنفسجي #7c5cff + سماوي #22d3ee + وردي #ff5c8a + تدرج --grad على خلفية #0a0a12)، (3) الكائن البطل **نواة ذكاء اصطناعي** (وليس حروف AB): نواة متوهجة بنفسجية + شبكة عصبية 64 عقدة فيبوناتشي بخطوط اشتباك + قفص wireframe + حلقتا بيانات + 3 حزم وردية تدور، (4) معلومات كاملة، (5) تباين عالٍ (scrim خلف كل نص).
- 11 فصلاً RTL بخط Cairo/Tajawal: هيرو + marquee خدمات → من أنا → ما أقدّمه (6 خدمات) → بوسترات ×3 → ستوري بورد ×2 (قهوة+عطر) → لاندنق ×3 → دراسة حالة سولارا (سكرين الموقع الحي بـ object-fit:cover top) → هويات ×3 → أبواب (متجر/يوتيوب/درايف/index.html) → ماذا أوفر لك (5 وعود) → CTA واتساب.
- موشن مرفوع: FOV keyframes مع Dolly-Zoom في فصل "ماذا أوفر لك" (fov 47)، Dutch Roll خفيف، كوريغرافيا إضاءة rim بنفسجي→سماوي→وردي عبر p، RGB Shift يركب سرعة التمرير، Scramble للعناوين الجانبية (مع setTimeout ينهيه إجبارياً خلال 1.1ث)، نبض النواة + "سوبرنوفا" في الختام، شريط تقدم علوي بالتدرج.
- **ترقيات v3 (حزم الترابط/المونتاج/القصة/التشطيب — بعد تحليل بروفيسور وافق عليه):**
  1. **الكاميرا مقيّدة بالفصول لا بالبكسلات:** `measure()` يقيس رؤوس الفصول من DOM (واعي بالـ pin-spacer) و`chapterP(sy)` يحوّل السكرول إلى (فصل + كسر بسموذستيب dwell) → `path.getPoint((i+f)/10)`. هذا يصلح انحراف التزامن على الجوال نهائياً. السكراب يدوي: `sy += (scrollY-sy)*0.055` في rAF.
  2. **Signal Pulses:** 16 نبضة ضوء تجري على خطوط الشبكة (سرعتها تزيد مع سرعة التمرير والفلاشات) — النواة "تفكر".
  3. **حالات لكل فصل:** مصفوفات ASIDE/DIM/SCALE (11 قيمة) مع lerp — الكانفس يعتم 0.62 في فصول القوائم (خدمات/أبواب) ليرتفع التباين، والبطاقات صارت شبه معتمة rgba(19,19,34,.88).
  4. **Look-at Offset:** الكاميرا تنظر `net.position*0.30` — الكائن في ثلث والنص في ثلث.
  5. **Fly-Through Cut:** غطسة gauss حول p=0.65 (قبل الهويات) تخترق الشبكة + سبايك RGB + اتساع FOV.
  6. **فصل البوسترات مثبّت (Pin):** `#adsPin` مع ScrollTrigger pin end:+=1500 — البرجر يتحول عادية→إعلان→بوستر أمامك، شرائح stgchips تتفعّل، وبجانبه مصغّرا العطر والأثاث. (صورتا food_orig/food_ad أُضيفتا للأصول = 15 صورة).
  7. **مشهد الذروة "ماذا أوفر لك":** 5 وعود سطور كبيرة تدخل واحداً واحداً وكل سطر يفجّر نبضة بالنواة عبر `window.ABFLASH()` (الخامس أقوى) + Dolly-Zoom.
  8. **تشطيبات:** أزرار CTA بالهيرو (شاهد أعمالي → #adsPin عبر lenis.scrollTo، تواصل معي → واتساب)، فهرس نقاط جانبي #dots (يختفي <900px)، أرقام شبحية ٠١-٠٤ + ✦ خلف الفصول، Camera Shake مع السوبرنوفا، على الجوال العمودي النواة تنزل تحت نص الهيرو (heroDrop) والإزاحة -1.05 بدل -2.8.
- الكائن يتنحى **يسار الشاشة دائماً** عبر متجه right من مصفوفة الكاميرا (وليس world-x).
- 15 صورة حقيقية data-URI في /tmp/folio_assets.json.
- **درس مهم:** كشف العناصر عبر IntersectionObserver + gsap.set/to وليس gsap.from+scrollTrigger (مع Lenis البطاقات تبقى opacity:0). الهيرو مستثنى لأن البريلودر يحرّكه. و`measure()` يجب أن يُستدعى قبل أول `loop()` وإلا انهار الرندر (undefined.x).
- المصدر: scratchpad/store/src/folio.js + build-film.py (يولّد HTML كاملاً ويحقن bundle). exposure 0.85، bloom 0.38، emissive 0.95 — كل فصل يجب أن يجلس في 100svh (صغّر مقاسات البطاقات لو انقص شيء).

## ab-gallery.html — «الممر المعماري» البورتفوليو التفاعلي (فكرة 2 منفّذة)
- صفحة مستقلة: https://abnewtry.github.io/desktop-tutorial/ab-gallery.html — متحف ليلي 3D **بلا تمرير إطلاقاً**: كل التنقل بالضغط (أزرار/لوحات/أسهم/سحف)، والكاميرا تتحرك مع تفاعل العميل.
- **8 قاعات** بممر واحد (HL=11 وحدة): من أنا → ما أقدّمه (6 لوحات خدمة) → إعلانات وبوسترات (3 + قصة التحول داخل بانل البرجر بصور المراحل الثلاث) → ستوري بورد (2 + الشرح الثلاثي) → لاندنق بيجز (3 + سولارا على الجدار الخلفي برابط الموقع الحي) → هويات (3) → أبواب المشاريع (متجر/يوتيوب/درايف بروابط خارجية) → الوعود (4 لوحات) + قطعة الختام «🧠 عقل مسوّق» ببانل التواصل الكامل وزر واتساب.
- **الاكتشاف gamified:** 27 قطعة، عدّاد «اكتشفت X/27» + شريط تقدم بالتدرج أعلى الشاشة، حلقة سماوية نابضة تحت كل قطعة غير مكتشفة تختفي عند فتحها، والتقدم محفوظ localStorage (مفتاح `abgal`).
- **التفاعل:** raycaster على اللوحات (hover يضيء إطارها + يكبر المؤشر)، الضغط يقرّب الكاميرا للقطعة (gsap 1.3s power3.inOut) ويفتح بانل جانبي (RTL يمين، وعلى الجوال bottom-sheet 56svh). أزرار السابق/التالي + اسم القاعة، أسهم كيبورد وسحب باللمس، Esc يغلق.
- **الجوال العمودي = جولة موجّهة:** الجدران الجانبية خارج مجال الرؤية الرأسي، لذلك `isPortrait()` يحوّل التالي/السابق/السحب إلى تنقّل قطعة-قطعة (`tourGo`) مع عداد «قاعة · قطعة X/27»، وزر الدخول يركّز أول قطعة تلقائياً.
- **المشهد:** جدران حجرية داكنة + أقواس بحواف نيون بنفسجية بين القاعات + أرضية Reflector رخام أسود (فوقها طبقة تعتيم 0.55) + مخاريط ضوء وهمية فوق البراويز + غبار عائق + PointLights بنفسجية كل قاعتين. الأعمال MeshBasic (غير متأثرة بالإضاءة = ألوان الصور صافية). bloom 0.22/0.92 (أعلى يحرق اللاندنق البيضاء).
- **دروس مهمة:** (1) البراويز تُدار لتواجه نقطة وقوف القاعة `atan2(dir.x,dir.z)` وإلا تظهر بزاوية حادة غير مرئية. (2) نقطة الوقوف داخل القاعة z0+5.0 وfov 46 — خلف القوس يحجب الجانبين. (3) `TextureLoader.load` غير متزامن: قصّ cover عبr callback التحميل وليس مباشرة. (4) لوحات النص canvas 512×640 بإطار متدرج (`plaqueTex`) وctx.direction='rtl'.
- المصدر: scratchpad/store/src/gallery.js + build-gallery.py (يبني HTML ويحقن window.ABASSETS + bundle). لا Lenis ولا ScrollTrigger — gsap فقط، الصفحة overflow:hidden. hooks للفحص: `window.__go(i)` و`window.__open(id)`.
