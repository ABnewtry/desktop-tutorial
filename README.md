# بورتفوليو — مصمم ومسوّق بالذكاء الاصطناعي / AI Designer & Marketer Portfolio

موقع بورتفوليو شخصي **ثنائي اللغة** (عربي RTL / إنجليزي LTR) لمصمم ومسوّق بالذكاء الاصطناعي:
إعلانات، بوسترات، لاندنق بيجز، مواقع، منيوهات، ومونتاج فيديو.

A **bilingual** (Arabic RTL / English LTR) personal portfolio for an AI-powered designer & marketer.

## ✨ المزايا / Features
- ملف واحد `index.html` بدون أي اعتماديات أو build — يفتح مباشرة.
- زر تبديل اللغة (عربي ⇄ إنجليزي) مع تبديل اتجاه الصفحة تلقائياً.
- وضع ليلي/نهاري (يُحفظ في المتصفح).
- تصميم متجاوب (موبايل-أولاً)، أنيميشن يحترم `prefers-reduced-motion`.
- أقسام: Hero / الخدمات / معرض الأعمال (تبويبات) / نبذة / تواصل.

## 🛠️ كيف تخصّصه / How to customize
كل المحتوى داخل `index.html` في كائن `DATA` (عربي + إنجليزي) وقائمة `SOCIALS`:
1. غيّر `brand` لاسمك، والنصوص في `ar` و `en`.
2. عدّل مصفوفة `work` وأضف صورك الحقيقية بدل التدرّجات.
3. حدّث روابط `SOCIALS` (إيميل، LinkedIn، GitHub، X، Instagram، WhatsApp).

## 🚀 النشر على GitHub Pages / Deploy
1. Settings → Pages → Source: `Deploy from a branch`.
2. اختر الفرع و `/ (root)` ثم Save.
3. الموقع يصير على `https://<username>.github.io/desktop-tutorial/`.
