# AB — محتوى الكولكشنات الثلاثة + برومبتات كلاود اكستنشن

> **القواعد المعتمدة:**
> - كل كولكشن = **منتجين فقط** (لا أكثر).
> - كل المنتجات **تيشيرتات** فقط.
> - **الفرونت بيج = بوستر فقط** (بدون عرض منتجات على الصفحة) + زر "تسوّق المجموعة".
> - **الترتيب:** 1) TRAVEL الترحال (البوستر جاهز ✅) → 2) ANIME أنمي → 3) COUPLES الثنائيات.
> - الطلب المسبق مؤجّل.
>
> **طريقة الاستخدام:** افتح كلاود اكستنشن (المتصل بشوبي فاي)، الصق البرومبت الخاص بكل كولكشن، وارفق صورة البوستر مع برومبت TRAVEL. كلاود ينشئ الكولكشن، يرفع البوستر، يضيف التيشيرتين بالأوصاف والأسعار والمقاسات، ويربطها.
>
> **الأسماء بالإنجليزي + الأوصاف بالعربي. كل المنتجات تبدأ Draft (مسودة).**
